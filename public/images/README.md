# 视觉素材放这里

沉浸式体验需要两张女神石像全屏背景图。文件名必须**完全一致**：

| 文件名 | 用途 | 说明 |
|--------|------|------|
| `goddess-entry.png` | 进入页全屏背景 | 端坐女神（正面半身） |
| `goddess-main.png` | 主交互页全屏背景 | 张开双手的女神（双手托起泡泡） |

要求与建议：
- 图片会以 `width:100vw; height:100svh; object-fit:cover; object-position:center` 全屏显示，
  建议横向 ≥1920px 的高清图。
- 白底即可（页面自带柔光处理）。
- 没放图也不报错——自动回退为梦境渐变底，放进去即自动生效。

上传方式：GitHub → 本目录 → Add file → Upload files → 拖入 → Commit。
