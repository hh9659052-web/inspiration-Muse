# 视觉素材放这里

封面页与进入页会用到两张女神像图片。请把你的图放进本目录，文件名必须**完全一致**：

| 文件名 | 用途 | 对应参考 |
|--------|------|---------|
| `cover-goddess.png` | 主封面（端坐女神） | 图一 |
| `entry-goddess.png` | 进入页（张开双手的女神） | 图二 |

建议：
- 透明背景 PNG（抠掉白底），这样能融进大理石/天空渐变背景。
- 高度方向分辨率 ≥ 1200px，竖向构图。
- 没放图也不会报错——页面会优雅地隐藏图片位、只显示文字与泡泡，放进去即自动生效。

水晶水泡（图四）已用 CSS 实现（`.bubble-crystal`），无需图片素材。
